💡 为什么要用这个规则？
🔤 大小写转换的原理：
• 在ASCII编码中，小写字母和大写字母有固定的数值关系
• 小写字母'a'的ASCII值是97，大写字母'A'的ASCII值是65
• 它们相差32：'a' - 'A' = 97 - 65 = 32
ASCII对照表（部分）
字符: a b c ... z
ASCII: 97 98 99 ... 122
字符: A B C ... Z
ASCII: 65 66 67 ... 90
差值: 32 (小写 - 大写 = 32)
🧮 算法原理：
• 从左到右遍历每个字符
• 如果是小写字母（'a'到'z'），就减去32转换为大写
• 如果不是小写字母，保持不变
🎯 为什么减去32？
• 因为小写字母的ASCII值比对应大写字母大32
• 例如：'h'的ASCII是104，'H'的ASCII是72，104-32=72
• 所以：小写字符 - 32 = 对应大写字符

📝 Java代码答案 - 小写转大写
    public static void exercice3() {
      Scanner sc = new Scanner(System.in);
      System.out.print("请输入字符串: ");
      String s = sc.nextLine();

      String result = "";

      // 从左到右遍历字符串
      for (int i = 0; i < s.length(); i++) {
          char ch = s.charAt(i);

          // 如果是小写字母，就转换为大写
          if (ch >= 'a' && ch <= 'z') {
              ch = (char)(ch - 32); // 'a' 和 'A' 的 ASCII 差是 32
          }

          result += ch; // 拼接到结果字符串
      }

      System.out.println("原字符串: " + s);
      System.out.println("大写后: " + result);
    }
