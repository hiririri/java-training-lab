💡 为什么要用这个算法？
🗑️ 字符删除的原理：
• 字符删除就是从字符串中移除所有指定的字符
• 例如：从"banana"中删除所有'a'，得到"bnn"
删除过程（以"banana"中删除'a'为例）
原字符串: b a n a n a
位置: 0 1 2 3 4 5
处理: 保留 删除 保留 删除 保留 删除
结果: b + n + n = "bnn"
🧮 算法原理：
• 从左到右遍历字符串中的每个字符
• 比较当前字符是否等于要删除的字符
• 如果不等于，添加到结果字符串；如果等于，跳过
🎯 为什么用结果字符串？
• 结果字符串用来存储删除后的字符
• 只添加不需要删除的字符，跳过要删除的字符
• 最终结果字符串就是删除指定字符后的新字符串
最终结果：删除字符 - 后得到 -
📝 Java代码答案 - 字符删除
    public static void exercice6() {
      Scanner sc = new Scanner(System.in);

      // 读取要删除的字符
      System.out.print("请输入要删除的字符: ");
      char target = sc.nextLine().charAt(0);

      // 读取字符串
      System.out.print("请输入字符串: ");
      String s = sc.nextLine();

      String result = "";

      // 遍历每个字符，若不是目标字符则保留
      for (int i = 0; i < s.length(); i++) {
          char ch = s.charAt(i);
          if (ch != target) {
              result += ch;
          }
      }

      System.out.println("原字符串: " + s);
      System.out.println("删除字符 '" + target + "' 后: " + result);
    }
