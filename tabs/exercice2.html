💡 为什么要用这个规则？
🔢 二进制转十进制的原理：
• 每个二进制位代表一个2的幂次：最右边是2⁰，向左依次是2¹、2²、2³...
位权表（以1010为例）
位值: 1 0 1 0
位权: 2³ 2² 2¹ 2⁰
数值: 8 4 2 1
结果: 1×8 + 0×4 + 1×2 + 0×1 = 10
🧮 位权算法原理：
• 每个二进制位都有对应的位权：最右边是2⁰，向左依次是2¹、2²、2³...
• 直接计算每位对最终结果的贡献：位值 × 2^位权
• 把所有位的贡献值相加，就是最终的十进制结果
🎯 为什么这样计算？
• 二进制数的每一位代表不同的权重
• 最右边的位权重最小（2⁰=1），最左边的位权重最大
• 直接按位权计算，逻辑清晰，容易理解

📝 Java代码答案 - 二进制转十进制
    public static void exercice2() {
      Scanner sc = new Scanner(System.in);

      System.out.print("请输入二进制数: ");
      String binary = sc.nextLine();

      int result = 0;
      boolean isValid = true; // 用于检查是否只含 0 和 1

      // 从左到右遍历每一位，使用位权算法
      for (int i = 0; i < binary.length(); i++) {
          char bit = binary.charAt(i);

          // 检查输入是否合法
          if (bit != '0' && bit != '1') {
              isValid = false;
              break;
          }

          int bitValue = bit - '0'; // '0'→0, '1'→1
          int power = binary.length() - 1 - i; // 计算位权
          int contribution = bitValue * (int)Math.pow(2, power); // 计算贡献值
          result = result + contribution; // 累加贡献值
      }

      if (isValid) {
          System.out.println("二进制: " + binary);
          System.out.println("十进制: " + result);
      } else {
          System.out.println("输入错误！只能包含 0 和 1。");
      }
    }
