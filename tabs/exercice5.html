计数规则：if (s[i] == target) count++;

💡 为什么要用这个算法？
🔍 字符计数的原理：
• 字符计数就是统计某个特定字符在字符串中出现的次数
• 例如：在"banana"中，字符'a'出现了3次
计数过程（以"banana"中找'a'为例）
字符串: b a n a n a
位置: 0 1 2 3 4 5
匹配: × ✓ × ✓ × ✓
计数: 0 → 1 → 1 → 2 → 2 → 3
🧮 算法原理：
• 从左到右遍历字符串中的每个字符
• 比较当前字符是否等于目标字符
• 如果相等，计数器加1；否则保持不变
🎯 为什么用计数器？
• 计数器可以累加记录匹配的次数
• 每次找到匹配的字符，计数器就增加1
• 最终计数器的值就是目标字符出现的总次数
最终结果：字符 - 在字符串中出现了 - 次
📝 Java代码答案 - 字符计数
    public static void exercice5() {
      Scanner sc = new Scanner(System.in);

      // 读取目标字符
      System.out.print("请输入要计数的字符: ");
      char target = sc.nextLine().charAt(0);  // 取第一个字符

      // 读取字符串
      System.out.print("请输入字符串: ");
      String s = sc.nextLine();

      int count = 0;

      // 从左到右遍历整个字符串
      for (int i = 0; i < s.length(); i++) {
          if (s.charAt(i) == target) {
              count++;
          }
      }

      System.out.println("字符 '" + target + "' 在字符串 \"" + s + "\" 中出现了 " + count + " 次");
    }
